CLASS PORTRAITS — drop your PNGs here.

The avatar shows a rank-based class portrait. Save each transparent PNG in
THIS folder with EXACTLY these filenames (lowercase):

  seer.png       (Lv 1+   — hooded mystic)
  vanguard.png   (Lv 15+  — steel knight)
  warden.png     (Lv 30+  — winged guardian)
  sovereign.png  (Lv 45+  — crowned enchanter)
  titan.png      (Lv 60+  — horned champion)
  samurai.png    (Lv 80+  — oni samurai warlord, apex)

Tips:
- Transparent background, square (1:1), composed for a circular crop
  (face + upper chest centred; nothing important in the corners).
- Any class without a file here automatically falls back to the built-in
  SVG avatar, so it's fine to add them one at a time.
- Level breakpoints live in src/data/classes.ts (change minLevel there).
